Terraforming Mars Slack bot live

This deployment is a serverless Slack app. It does not serve a web UI. If you reached this page in a browser, the function is up and Slack can reach it.

  • Slack endpoint: /api/slack/events (POST only)
  • Slash command: /tm-newgame from any channel in the workspace where this app is installed.
  • Source & setup docs: see the slack-bot/ directory in the terraforming-mars repo (your fork).
